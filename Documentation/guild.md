# GET /guild
> Use this endpoint to get information about a guild

### Query Needed
* `guildid` - Guild ID
### Parameters Needed
* None


## BDFD Example
````ruby
$httpGet[$getVar[apiURL]/guild?guildid=$guildID]
$description[```json
$httpResult```]
````
