# GET /guild/roles
> Use this endpoint to get the roles of the guild

### Query Needed
* `guildid` - The guild id
### Parameters Needed
* None

## BDFD Example
````ruby
$httpGet[$getVar[apiURL]/guild/roles?guildid=$guildID]]
$description[```json
$httpResult```]
````
