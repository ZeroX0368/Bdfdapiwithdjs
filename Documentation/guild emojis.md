# GET /guild/emojis
> Use this endpoint to get guild emojis

### Query Needed
* `guildid` - The guild id
### Parameters Needed
* None

## BDFD Example
````ruby
$httpGet[$getVar[apiURL]/guild/emojis?guildid=$guildID]]
$description[```json
$httpResult```]
````
